HTML lista tagova:

< h1 > do < h6 >
Heading, koristi se naslove, a zavisno od indeksa uz slovo h smanjuje se i veličina teksta u naslovu.
< p >
Paragraf.
< hr >
Koristi se za tematsko odvajanje cjelina. U suštini predstavlja horizontalnu liniju.
< abbr >
Definiše skraćenicu ili akronim.
< ol >
Orderd list, predstavlja listu koja je numerisana brojevima ili slovima.
< ul >
Unorderd list, predstavlja listu koja je predstavljena tačkicama.
< li >
List item, predstavlja element liste i koristi se kao dijete prethodna dva.
< bdo >
Ispisivanje tekta unazad.
< button >
Dugme koje se može kliknuti.
< pre >
Preformatted text, prikazuje tekst u browseru tačno onako kako smo ga ispisali u okviru ovog taga.
< a >
Another ili ti sidro je tag koji definiše hyperlink. Koristi se sa atributom href, kojim se dofiniše lokacija tog hyperlinka.
< iframe >
Inline frame se koristi za ugrađivanje drugog dokumenta unutar našeg HTML dokumenta.
< em >
Emphasized text iskrivi naš tekst, vizuelno isto kao i italic.
< strong >
Strong text podeblja naš tekst, vizuelno isto kao i bold
< !DOCTYPE >
Prva stvar koja se radi kod pisanja HTML dokumenta. Govori browseru koji tip dokumenta da očekuje
< html >
Govori browseru da je riječ o HTML dokumetu. U njega idu svi ostali tagovi osim < !DOCTYPE >
< body >
Predstavlja tijelo dokumenta, sve što se ispiše unutar Body taga biće prikazano u browswru.
< head >
Ovaj tag definiše glavu dokumenta i svi elementi ovog taga neće se prikazati u browseru.
